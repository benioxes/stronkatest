# NexaStudio - Creative Agency Website

## Overview
A modern, professional website for a fictional creative digital agency called "NexaStudio". This is a portfolio-ready website designed to showcase web development skills and attract potential clients.

## Project Structure
```
/
├── server.js              # Express server
├── public/
│   ├── index.html         # Main HTML file
│   ├── css/
│   │   └── style.css      # All styles
│   └── js/
│       └── main.js        # Interactive features
├── package.json           # Node.js dependencies
└── replit.md              # Project documentation
```

## Technologies Used
- **Backend**: Node.js with Express
- **Frontend**: HTML5, CSS3 (custom), Vanilla JavaScript
- **Fonts**: Google Fonts (Poppins)
- **Icons**: Font Awesome 6

## Features
- Responsive design (mobile, tablet, desktop)
- Animated hero section with floating gradient orbs
- Statistics counter animation
- Services grid with hover effects
- Portfolio gallery with filter functionality
- Testimonials slider with auto-rotation
- Contact form with animations
- Smooth scroll navigation
- Mobile hamburger menu

## Running the Project
```bash
node server.js
```
The server runs on port 5000.

## Sections
1. **Hero** - Main landing with CTA and statistics
2. **Services** - 6 service cards with features
3. **Portfolio** - Filterable project gallery
4. **About** - Team and values section
5. **Tech Stack** - Scrolling technology carousel
6. **Testimonials** - Client reviews slider
7. **CTA** - Call-to-action banner
8. **Contact** - Form and contact information
9. **Footer** - Links and newsletter signup
